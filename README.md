hamecomic
=========

"hamecomic" is a comic-like picture frame for your pictures.


Installation
-
    git clone git@github.com:pbl-cloud/hamecomic.git
    cd hamecomic/pkg
    gem install -l hamecomic-0.1.0.gem


Usage
-

    $ hamecomic [mode] [input_file]

[mode] : Choose from { kansha, mada, keikaku}.
